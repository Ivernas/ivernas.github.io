6.2 int

和C/C++类似，
uint8、uint16、uint32、uint64分别是无符号的8位、16位、32位、64位，
int8、int16、int32、int64则是有符号的。
byte : 8位的数据
rune : int32
int : 看操作系统、32位或64位有符号
uint ：看操作系统、32位或64位有符号
uintptr : 用于存放指针

6.3 float

float32 ：IEEE-754 32位浮点型数
float64 ：IEEE-754 64位浮点型数
complex64 : 32位实数和32位虚数
complex128 : 64位实数和64位虚数

6.5 指针

概念和用法和C/C++中差不多。
不同点：

  1. go中空指针用nil表示。
  2. 指针不能用来运算，即不能通过加减操作指向附近的地址。
  3. 没有->这种语法，可以用*p表示指向的对象，或者直接用.访问结构体中的成员。

6.7 切片(slice)

使用方法和python中切片差不多，用起来就是长度不定，可以增删的数组。
看初始化方法和使用方法，猜测内部实现是对数组进行了一些引用和组织，内部记录了在数组中起始和终止对应的index,长度不够时生成新的数组以此来扩充容量，类似于C++中的vector的实现。
