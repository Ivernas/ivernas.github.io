4 变量和常量

同C/C++，名称规则为数字、字母和下划线，不能以下划线开始。

4.1 变量声明和赋值

go里没有声明和定义的分别，声明时就定义了这个变量。声明时不赋值，使用默认值。
①用var关键字声明，类型写在变量名后面
例：var isSuccess bool
②不用写变量类型，编译器根据赋值类型推断。
例：var isSuccess = true
③声明时省略var 用:=实现声明加初始化，如果变量已经声明过，不能用这种形式。
*只能用在函数体内，不能用在全局变量的声明初始化
例: isSuccess := true
